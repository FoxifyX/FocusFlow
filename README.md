# FocusFlow ⏱️✨

A **modern, multi-functional productivity desktop app** built with pure Python and Tkinter — inspired by Material Design.

FocusFlow helps you manage **multiple timers, track focus tasks, use a stopwatch, and stay motivated with quotes** — all in one clean interface.

---

## 🚀 Features

### ⏳ Multi Timer System
- Create **multiple timers** (study, workout, breaks, etc.)
- Each timer runs **independently**
- Start / Pause / Reset anytime
- Visual **progress bar** for each timer
- Auto-detects when a timer is completed
- Timers continue even after closing the app ✅

---

### 💾 Persistent Storage
- All data is saved locally (`.focusflow_data.json`)
- Timers **resume automatically after reopening**
- Theme and tasks are also saved

---

### ⏱️ Stopwatch
- Start / Pause / Reset stopwatch
- Real-time precision tracking
- Clean large display

---

### 🎯 Focus Mode
- Save your **current task**
- Keeps you focused on one goal
- Displays your task clearly on screen

---

### 💡 Motivation Quotes
- Random inspirational quotes
- One click to refresh
- Helps maintain productivity mindset

---

### 🎨 Modern UI
- Light & Dark mode toggle 🌙
- Clean **Material-inspired design**
- Smooth layout with tabs:
  - Timers
  - Stopwatch
  - Focus
  - Quotes
